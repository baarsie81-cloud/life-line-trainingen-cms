# Life-Line-Trainingen website

Nieuwe website voor Life-Line-Trainingen, gebouwd met Astro en Tailwind CSS.

## Ontwikkelen

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Het project is klaar voor deployment via Vercel. Koppel deze GitHub-repository aan Vercel en gebruik de standaard Astro-instellingen.
